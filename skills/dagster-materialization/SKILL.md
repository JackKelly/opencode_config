# Dagster Materialization Workflow

## What I do
- Provide instructions for materializing Dagster assets from the CLI.
- Ensure the materialization command is executed using the correct syntax.
- Syntax: `dg launch --assets "<asset_name>"`

## Workflow
1.  Identify the asset name you wish to materialize.
2.  Use the `bash` tool to run the command: `dg launch --assets "<asset_name>"`
3.  Check the output to ensure the materialization was successful.

## When to use me
Use this skill whenever you need to manually trigger the materialization of a Dagster asset from the command line.
